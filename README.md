# Three-Tier DevOps Project

## Overview
3-tier architecture application deployed on AWS EC2 using Docker, with Prometheus & Grafana monitoring, and CI/CD via GitHub Actions.

## Folders
- `backend/` - Node.js app
- `terraform/` - AWS infrastructure scripts
- `prometheus/` - Prometheus config
- `README.md` - Project description

## Features
- Dockerized backend
- Monitoring with Prometheus & Grafana
- CI/CD with GitHub Actions
- Free Tier AWS friendly
